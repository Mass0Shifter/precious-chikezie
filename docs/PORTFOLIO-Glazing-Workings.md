# Glazing Calculator (Workings) — Portfolio Notes

**(NDA)** Proprietary construction estimation tool for the glass industry (client: **Leads Glazing**).

## Summary

A Flutter-based app with a **direct manipulation** UI: users compose glazing layouts visually via drag-and-drop. The system calculates materials, labor, and margins against a component library.

## Hosting & infrastructure

- **AWS** is used for **hosting** the production-facing stack (APIs, web assets, and deployment pipeline), supporting reliable delivery and scalable capacity as usage grows.

## Tech

- **Flutter** — mobile / cross-platform client  
- **React** — supporting web surfaces where applicable  
- **Node.js** — backend services  
- **Algorithms** — estimation and layout logic  
- **AWS** — hosting and cloud operations  

## Role

- Lead development, Flutter architecture, algorithm design  
- AWS deployment and hosting
